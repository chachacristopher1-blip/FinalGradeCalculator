# Final Grade Calculator

**Learning Project #1**

This is my first Java project uploaded to GitHub.

It was created as part of my programming learning journey, with the goal of improving my Java skills and building a public portfolio.
## Description
Final Grade Calculator is a simple Java console application that calculates a student's final grade using the grades from the first term, second term, and the final exam.

The program validates the input to ensure all grades are between 0 and 10 before calculating the final result.

## Features
- Input validation (grades must be between 0 and 10).
- Calculates the final grade.
- Displays whether the student passed or failed.
- Simple console interface.

## Technologies
- Java
- Apache NetBeans
- Maven

## How to Run
1. Open the project in Apache NetBeans.
2. Build the project.
3. Run the application.
4. Enter the requested grades.
